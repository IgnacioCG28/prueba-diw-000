Sección 1 - Descripcion definición Accesibilidad web.

La accesibilidad web se refiere a la capacidad de permitir que todas las personas, sin importar sus limitaciones o discapacidades, tengan acceso a la World Wide Web y a sus contenidos. En esencia, se trata de un diseño web que posibilita que estas personas puedan percibir, comprender, navegar y interactuar con la Web, a la vez que facilita la creación de contenidos accesibles. Además de beneficiar a aquellos con discapacidades, la accesibilidad web también mejora la experiencia de uso para personas mayores cuyas habilidades pueden verse afectadas por el envejecimiento. La accesibilidad web abarca una amplia gama de discapacidades que pueden afectar el acceso a Internet, como las discapacidades auditivas, cognitivas, neurológicas, físicas, del habla y visuales. No obstante, sus beneficios se extienden más allá de las personas con discapacidad, ya que también facilita la navegación y la interacción en dispositivos con pantallas pequeñas, diferentes modos de entrada, como teléfonos móviles, relojes inteligentes y televisores inteligentes. Para los desarrolladores y organizaciones que desean crear sitios web y herramientas de alta calidad, la accesibilidad web es esencial. Esto les permite no excluir a ninguna persona del acceso a sus productos y servicios. Además, en muchas ocasiones, la accesibilidad web es un requisito legal. El objetivo fundamental de la accesibilidad web es garantizar que las páginas web sean utilizables por la mayor cantidad de personas posible, sin importar su nivel de conocimiento o habilidades personales, ni las características técnicas de los dispositivos utilizados para acceder a Internet.
